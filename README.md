# louisfoley.com — portfolio

Personal portfolio site for **Louis Foley Jr.**, Mechanical Engineer (Northeastern '26).
Static HTML/CSS/JS — no build step. Served by GitHub Pages from `main`.

## Pages
- `index.html` — about, skills, experience
- `projects.html` + `project-*.html` — project index and detail pages
- `resume.html` — resume + PDF download
- `contact.html` — contact
